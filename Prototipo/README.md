# Prototipo de Interfaz (TechZone Gaming)

Capturas de pantalla del diseño de interfaz para la plataforma, orientada a computadoras.

## Vistas Diseñadas
* **Landing Page**: Interfaz de bienvenida con buscador de productos y ofertas destacadas.
* **Catálogo y Filtros**: Visualización de categorías (Consolas, PC, Periféricos) y estados de stock.
* **Panel Administrativo**: Dashboard que resume el inventario actual gestionado en memoria, permitiendo cambios inmediatos sobre los productos disponibles.
