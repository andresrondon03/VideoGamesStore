# Prototipo de Interfaz - TechZone Gaming

Diseños visuales de la interfaz de usuario creados para validar la experiencia de navegación y la arquitectura de información.

## Vistas Principales
* **Página de Inicio**: Banner de "Tu Tienda Gaming Definitiva", buscador central y sección de ofertas especiales.
* **Catálogo**: Tarjetas de productos con visualización de descuentos (ej. -20%), precios y estado de stock.
* **Exploración por Categoría**: Acceso rápido a secciones de Monitores, Periféricos, Consolas y Realidad Virtual.
* **Panel de Administración**: Dashboard con métricas de valor de inventario, productos totales y tabla de gestión CRUD con acciones rápidas.
