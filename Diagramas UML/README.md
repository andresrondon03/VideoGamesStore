# Diagramas UML

Esta carpeta contiene el modelado visual del sistema **VideoGamesStore**, utilizando el Lenguaje de Modelado Unificado para definir la estructura y comportamiento del software.

## Contenido
* **Diagrama de Contexto**: Representación de los límites del sistema y su interacción con actores externos.
* **Diagrama de Clases**: Estructura estática que detalla entidades como `Usuario`, `Videojuego`, `Carrito` y patrones de diseño (Estrategia de Pago y Promociones).
* **Diagrama Entidad-Relación**: Modelo lógico de datos que define las tablas y relaciones en la base de datos.
* **Diagramas de Casos de Uso**: Flujos de procesos para registro/login, filtrado de productos, compra y gestión administrativa.
* **Diagramas de Secuencia**: Interacción temporal de objetos para funciones críticas como el checkout, gestión de inventario y autenticación.
